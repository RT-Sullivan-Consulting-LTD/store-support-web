Privacy Policy

Last updated: 16 June 2026

Sessionz.ai is developed by RT Sullivan Consulting Ltd, United Kingdom.

Data we collect

Sessionz.ai collects the minimum data needed to operate the AI plan generation feature:

  • Goals text and equipment list — the text you enter when generating a plan is sent to our plan generation server (Supabase Edge Functions) and forwarded to the Claude AI API (Anthropic). No personal identifiers are attached to this request. The text is used only to generate your plan and is not retained.
  • Subscription status — we receive confirmation from Apple that your subscription is active. We store only whether you have an active subscription and when it expires. No payment data is stored or accessible to us.
  • Anonymous user ID — Supabase assigns an anonymous identifier used solely to enforce per-user plan generation limits. This ID is not linked to your name, email, or Apple ID.

Data stored on your device

Your workout plans, session logs, set records, and personal profile (goals, equipment, unit preferences) are stored privately on your iPhone using SwiftData. This data:

  • Never leaves your device except as part of the plan generation request described above
  • Is not uploaded to any server or analytics service
  • May be included in your local iTunes/Finder backup or iCloud device backup — both controlled entirely by you
  • Is deleted when you uninstall the app

HealthKit

Sessionz.ai requests read access to Apple Health to optionally incorporate your recent activity data into plan generation. HealthKit data:

  • Is read only at the moment you tap Generate Plan, if you have granted permission
  • Is sent as part of the plan generation request and is not retained
  • Is never shared with third parties or used for advertising

You can revoke HealthKit access at any time in Settings → Privacy & Security → Health → Sessionz.

Third-party services

Sessionz.ai uses the following third-party services:

  • Supabase — hosts the plan generation API. Processes goals text and equipment list only. No personal data is stored on Supabase beyond the anonymous user ID and subscription expiry date.
  • Anthropic (Claude API) — receives goals text and equipment list to generate workout plans. Anthropic's data usage policy applies to this data; plans are not used to train models under the API terms.
  • Apple StoreKit — manages subscription payments. All billing is handled entirely by Apple.

No analytics SDKs, advertising networks, or crash reporters are included in the app.

Children

Sessionz.ai is not directed at children under 13. We do not knowingly collect data from children.

Changes to this policy

If this policy changes, the updated version will be published at this URL and the date above will be updated. We will not introduce new data collection practices without updating this policy.
